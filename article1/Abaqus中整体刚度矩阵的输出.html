Abaqus中整体刚度矩阵的输出

 

前两篇分别介绍了ANSYS中导出整体刚度矩阵的两种方法，忽然念及之前也曾尝试过从Abaqus中导出结构的整体刚度矩阵，故一并总结出来。

Abaqus中整体刚度矩阵的导出需要通过修改Input文件完成。建立模型完成后，在Job模块新建任务，选择“Write input”即可得到模型的Input文件。

 

Input文件是具有严格格式的ASCII码文本文档，为了输出整体刚度矩阵，可在文件最后增加如下一段命令：

** ----------------------------------------------------------------
** 
** STEP: Step-MatrixOutput
** 
*Step, name=Step-MatrixOutput
*Matrix Generate, stiffness, mass, viscous damping, structural damping, load
*Matrix Output, stiffness, mass, viscous damping, structural damping, load, format=Matrix Input
** 
** BOUNDARY CONDITIONS
** 
** Name: BC-1 Type: Displacement/Rotation
*Boundary
_PickedSet4, 1, 1
_PickedSet4, 2, 2
** 
** LOADS
** 
** Name: Load-1   Type: Concentrated force
*Cload
_PickedSet5, 1, 1000.
** 
*End Step

第五行*Step, name=Step-MatrixOutput 定义分析步名称，“Step-MatrixOutput”即为分析步名称，可自由定义。第六行*Matrix Generate, stiffness, mass, viscous damping, structural damping, load生成所关心的矩阵，“stiffness”“mass”“viscous damping”和“structural damping”表示生成整体刚度、质量、粘滞阻尼和结构阻尼矩阵，“load”表示生成荷载向量。第七行*Matrix Output, stiffness, mass, viscous damping, structural damping, load, format=Matrix Input用于输出第六行命令定义的整体矩阵与向量，其中format=Matrix Input指定最终输出的文件格式。一般情况下，采用“Matrix Input”和“Coordinate”较为方便，这里采用“Matrix Input”格式。第12~14行施加边界条件，一般可直接复制模型中已定义的边界条件。第19~20行定义外荷载，直接复制模型中定义的荷载即可。最后第22行*End Step结束分析步。

Abaqus的Input文件中，命令均以一个星号（*）开头，两个星号（**）开头的则为注释。另外需注意，Input文件中不能有空行。

从修改后的Input文件创建任务，如下图所示

并执行分析任务，可在当前工作目录下看到一系列以“.mtx”结尾的文件，这些文件即为导出的结构整体矩阵。文件命名格式一般为“Job-name_Matrix-name.mtx”。“Job-name”为分析任务名，以上图为例，则应为“Job-1”。“Matrix-name”则可能为“STIF2”“Job-1_MASS2”“DMPV2”“DMPS2”和“LOAD2”，分别表示整体刚度矩阵、整体质量矩阵、整体粘滞阻尼矩阵、整体结构阻尼矩阵和整体荷载向量。与ANSYS HBMAT命令输出的矩阵相同，这里得到的文件中同样只包含矩阵的下三角元素，故矩阵上三角元素需根据对称关系补全。

打开“Job-1_STIF2.mtx”可以看到文件具有以下格式：

xxxxxxxxxx
1,1, 1,1,  1.000000000000000e+36
1,2, 1,1,  3.750000000000001e+10
2,1, 1,1, -6.346153846153885e+10
2,2, 1,1, -2.884615384615387e+09
12,1, 1,1,  1.153846153846116e+10
12,2, 1,1,  2.884615384615388e+09
13,1, 1,1, -5.192307692307655e+10
13,2, 1,1, -3.750000000000001e+10
......

其中第一列为行节点号，第二列为行节点自由度编号，第三列为列节点编号，第四列为列节点自由度编号，第五列为矩阵元素。根据节点和自由度编号，可以计算矩阵非零元素的位置。由于该模型为二维模型，且仅有平动自由度，故每个节点仅有两个自由度。因此，第5行说明刚度矩阵中第(12−1)×2+1=23 行第(1−1)×2+1=1 ​列的元素取值为1.153846153846116e+10。这里需要注意的是，与ANSYS中不同，Abaqus中的边界条件采用加/乘大数法，因此被约束自由度对应的刚度矩阵元素会被设定为1.0×1036 ，如上例中的第一个元素。

若*Matrix Output命令中指定按照“Coordinate”格式输出矩阵和向量，则最终得到的文件格式如下：

xxxxxxxxxx
1 1  1.000000000000000e+36
1 2  3.750000000000001e+10
2 1  3.750000000000001e+10
1 3 -6.346153846153885e+10
3 1 -6.346153846153885e+10
1 4 -2.884615384615387e+09
4 1 -2.884615384615387e+09
1 23  1.153846153846116e+10

此格式更加易于读取：第一列为行编号，第二列为列编号，第三列为矩阵非零元素。

了解文件格式后，可编写程序读取文件中的矩阵。以下给出读取“Matrix Input”格式文件的MATLAB函数：

​x
function strucMCKMat = matreader_abaq2matlab(fileName, option)
% % a function to read mass/damping/stiffness matrix from output of Abaqus
%=================================================================================================
% % By Yang, Jia-Shu
% % Date: 2021-03-07
%=================================================================================================
% Input variables:
%----------------------------------------------------
% fileName : a string indicating which file to read.
%=================================================================================================
% Output variables:
%----------------------------------------------------
% strucMCKMat: mass/damping/stiffness matrix
%=================================================================================================
    if nargin == 1
        option = 'sparse';
    end


    if ~ischar(fileName) && ~isstring (fileName)
        error('The input of function  "matreader_abaq2matlab" should be a string! ');
    end
    
    %read all data from the file indicated by the input
    dataAll = readmatrix(fileName, 'FileType', 'text');
    
    %number of node
    idNodeRow   = dataAll(:,1);
    idNodeCol   = dataAll(:,3);
    
    %number of degree of freedom
    idDofRow    = dataAll(:,2);
    idDofCol    = dataAll(:,4);
    
    %non-zero matrix entry
    nonZeroEntry  = dataAll(:,5);
    nNonZeroEntry = length(nonZeroEntry);
    
    %lables and number of nodes
    idNodeUniRow = unique(idNodeRow);
    nNodeRow     = length(idNodeUniRow);
    idNodeUniCol = unique(idNodeCol);
    nNodeCol     = length(idNodeUniCol);
    if nNodeRow ~= nNodeCol
        error('The number of nodes is not compatiable!');
    end
    
    %the number of degrees of freedom of each node
    nDofEachNodeRow = zeros(nNodeRow, 1);
    nDofEachNodeCol = zeros(nNodeRow, 1);
    parfor ii = 1:nNodeRow
        nDofEachNodeRow(ii) = max(idDofRow(idNodeRow == idNodeUniRow(ii)));
    end
    parfor ii = 1:nNodeCol
        nDofEachNodeCol(ii) = max(idDofCol(idNodeCol == idNodeUniCol(ii)));
    end
    
    %the location of each entry
    locRow = zeros(nNonZeroEntry, 1);
    locCol = zeros(nNonZeroEntry, 1);
    parfor jj = 1:nNonZeroEntry
        locRow(jj) = sum((idNodeUniRow < idNodeRow(jj)) .* nDofEachNodeRow) + idDofRow(jj);
        locCol(jj) = sum((idNodeUniCol < idNodeCol(jj)) .* nDofEachNodeCol) + idDofCol(jj);
    end
    
    %Construct matrix
    strucMCKMat = sparse(locRow, locCol, nonZeroEntry);
    
    if strcmpi(option, 'full')
        strucMCKMat = full(strucMCKMat);
    end


    %Make a symmetrical matrix
    strucMCKMat1 = strucMCKMat + strucMCKMat';
    strucMCKMat1(logical(eye(size(strucMCKMat)))) = diag(strucMCKMat);
    strucMCKMat = strucMCKMat1;


end
